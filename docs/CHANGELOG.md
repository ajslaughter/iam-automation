# Changelog

## [0.1.0] - 2024-01-01
### Added
- Initial release of the IAM Lab Automation Toolkit.
- PowerShell module with helpers for OUs, security groups, users, and group membership maintenance.
- Environment bootstrap, organization build, bulk provisioning/deprovisioning, and reporting scripts.
- Sample configuration and CSV templates.
- Pester test suite and GitHub Actions workflow.
